# Known Issues

- Microphone selection is not fully routed through the capture backend in all cases yet
- Qwen3 ASR is slower than Parakeet v3 and is not expected to match Parakeet latency
- Transform-selected-text behavior still depends on app-specific selection and paste behavior in the target app
- Public distribution is still unsigned/notarization-free from an Apple release perspective, so first-run friction remains higher than a notarized app
