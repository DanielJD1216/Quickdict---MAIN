# Roadmap

## Near Term

- Improve microphone device routing so selected devices are used by capture, not just listed in UI
- Add more robust transform replacement across a wider set of macOS apps/editors
- Improve Qwen ASR install flow and model-state messaging
- Add stronger automated tests for model detection, transform behavior, and error handling

## Mid Term

- Add launch-at-login support
- Improve lock-mode UX and long-form dictation handling
- Add more language-aware behavior to the Qwen ASR path
- Add more robust release automation for signed distributables

## Longer Term

- Streamed transcription for lower end-of-speech latency
- Better app-state diagnostics and debug tooling
- Apple notarization and smoother public distribution
