# Changelog

## v1.0.1

- Fixed DMG-installed app access to local Ollama on `localhost`
- Improved selected-text transforms for local Qwen 3.5 models
- Expanded spoken list parsing for grocery lists, to-do lists, and multilingual list cues

## v1.0.0

- Added working local dictation pipeline with Parakeet v3
- Added real ASR model switching between Parakeet v3 and Qwen3 ASR
- Added raw-audio capture, end-of-session conversion, and diagnostics
- Added global trigger key support with presets and custom key combos
- Added transform-selected-text flow using local Ollama models
- Added separate model management for ASR models and transform models
- Added DMG packaging and installation documentation
